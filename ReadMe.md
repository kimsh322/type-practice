## 타자 연습 게임

    간단한 타자 연습 게임 입니다.

    화면에 일정 시간 간격으로 랜덤한 위치에 랜덤한 단어가 생성됩니다.

    하단의 input박스에 단어를 입력하고 엔터를 누릅니다.
    일치하는 단어가 있으면 그 단어가 화면에서 지워집니다.

### 완성된 기능

    화면에 랜덤하게 단어 뿌리는 기능
    단어 입력시 단어 삭제 기능

### 추가할 기능

    시간이 지나면 단어 나오는 속도가 빨라지는 기능
    화면에 존재하는 단어가 15개 이상이 되면 Game Over 되는 기능
    시작 기능 - 시작버튼 누르면 이름 적고 시작, 시작하면 시작버튼 사라짐
    랭킹 기능
    애니메이션 효과
